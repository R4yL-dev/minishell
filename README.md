>### Installation:
>
>1. First, git clone the repository:
>   - `git clone https://github.com/Ray-cmd/minishell Minishell_42`
>1. Then, compile the project at the root of the repository:
>   - `make`
>1. Finally, launch the minishell:
>   - `./minishell`
